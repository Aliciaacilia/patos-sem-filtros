# Patos Sem Filtros

**Patos Sem Filtros** é uma plataforma para **registro de denúncias**, desenvolvida como projeto acadêmico da disciplina de **Lab. de Engenharia de Software do curso de Ciência da Computação da Universidade Estadual da Paraíba (UEPB), Campus VII**. Trata-se de um sistema Web que permite aos moradores da cidade de Patos, PB denunciar descasos e relatar problemas relacionados à infraestrutura urbana da cidade. O Patos Sem Filtros utiliza o conceito de feed dinâmico para ampliar a visibilidade das reclamações, facilitando o contato com os setores responsáveis e incentivando a participação cidadã ativa.

---

## Tecnologias Utilizadas

- Java; 
- JavaScript; 
- Node.js;  
- PostgreSQL;  
- Git & GitHub;
- HTML5 e CSS.

---

## Funcionalidades Atuais

- Cadastro e Login de Usuários/Moradores;

- Registro de denúncias;

- Feed Geral de denúncias;
  
- Histórico de usuário.

- Adicionar curtidas e comentários
